# Humanizer 🖊️

A free, unlimited AI text humanizer. No subscriptions. No word limits.

## Features
- 100% local mode with Ollama (nothing leaves your computer)
- Cloud mode with Anthropic API
- 5 writing styles — Casual, Academic, Professional, Conversational, Storytelling
- Light / Moderate / Heavy rewrite intensity
- Works on Windows, Mac and Ubuntu
- Single HTML file — no install needed

## How to run
Just open humanizer.html in any browser.

### Local mode (Ollama)
Install Ollama from ollama.com, then:
# Mac/Linux
OLLAMA_ORIGINS=* ollama serve

# Windows PowerShell
$env:OLLAMA_ORIGINS="*"
ollama serve

### Cloud mode
Get a free API key from console.anthropic.com and paste it in the app.
